ก้าวที่ 1: สมาชิกใหม่ (Day 1)

ปฐมนิเทศ — เรียนรู้วัฒนธรรมองค์กร (PKG 6 รับ, 9 มาตรฐาน, 20 วัฒนธรรม), เปิด Account ระบบ, กำหนด IDP

🏢 ทุนองค์กร 👤 ทุนมนุษย์ 💻 ทุนสารสนเทศ
📚

ก้าวที่ 2: อบรมปฐมนิเทศ (เดือนที่ 1)

เข้าอบรม: LIA สมาชิก PKG, Cost Down PKG, ความปลอดภัย, การใช้ AI เพื่อการทำงาน — ประเมินทักษะเริ่มต้น

🏢 LIA + Cost Down 👤 ความปลอดภัย 💻 AI พื้นฐาน
🌱

ก้าวที่ 3: พัฒนาทุนมนุษย์ (เดือนที่ 2-3)

พัฒนาทักษะการสื่อสาร, การทำงานเป็นทีม, การแก้ปัญหาพื้นฐาน — เข้าอบรมหลักสูตรประถม

👤 การสื่อสาร 👤 ทีมเวิร์ค
💼

ก้าวที่ 4: ทำงานจริง (เดือนที่ 3-6)

ปฏิบัติงานจริง — เก็บชั่วโมงอบรม, สะสมหน่วยกิต, พัฒนา GPA — ผ่านประเมินประถม

🏢 กระบวนการทำงาน 💻 เครื่องมือพื้นฐาน
✅

ก้าวที่ 5: ผ่านประเมิน (เดือนที่ 6)

ผ่านเกณฑ์: หน่วยกิต ≥ 44, GPA ≥ 3.5, ผ่านที่ 3 — ได้รับใบรับรองระดับประถม

🏢 ใบรับรอง
🎓

ก้าวที่ 6: อบรมทุติยนิเทศ (เดือนที่ 6-12)

เข้าอบรมระดับทุติย: กลยุทธ์องค์กร, ภาวะผู้นำ, AI ขั้นสูง, Data Analysis

🏢 กลยุทธ์ 👤 ภาวะผู้นำ 💻 AI ขั้นสูง
🤝

ก้าวที่ 7: พี่เลี้ยง (Mentor) (ปีที่ 1-2)

เป็นพี่เลี้ยงให้สมาชิกใหม่ — Coaching, Mentoring — พัฒนาผู้อื่น

👤 Coaching 👤 Mentoring

ก้าวที่ 8: อบรมตติยนิเทศ → จตุรนิเทศ → ปัญจนิเทศ

💡 UX Guide: ทำไมใช้งานง่าย?ออกแบบตามหลัก Human-Centered Design

🎯 หลักการ UX ที่ใช้

1. เห็นภาพรวมใน 3 วินาที

Dashboard แสดง KPI หลัก 4 ตัว — สมาชิก, ผ่านเกณฑ์, หลักสูตร, ชม.อบรม

2. สีสื่อความหมาย

เขียว = ผ่าน, เหลือง = รอ, แดง = ไม่ผ่าน — ไม่ต้องอ่านก็เข้าใจ

3. จัดกลุ่มตามงาน

ADMIN จัดการ / สมาชิก ดูผล — แยกมุมมองชัดเจน

4. กดน้อยสุด = ทำได้มากสุด

1 คลิก → ดูรายละเอียด, 2 คลิก → อนุมัติ, 3 คลิก → ส่งรายงาน

🚀 สรุป: ทำไมเลือกใช้ Layout 2 (Top Nav)

Layout 2 (Top Navigation) เป็นตัวเลือกที่ดีที่สุดสำหรับ PP5 เพราะ:

✅ เหมาะกับทั้ง ADMIN และสมาชิก — เมนูไม่เยอะเกินไป
✅ รองรับมือถือได้ดี — Top Nav ซ่อนใน Hamburger Menu ได้
✅ ขยายได้ง่าย — เพิ่มเมนูได้โดยไม่รก
✅ เรียนรู้เร็ว — ผู้ใช้คุ้นเคยกับ Top Nav จากเว็บทั่วไป

แนะนำ: ใช้ Layout 2 เป็นหลัก + เพิ่ม Sidebar แบบ Collapsible สำหรับ ADMIN PAD เมื่อต้องการจัดการเยอะ
