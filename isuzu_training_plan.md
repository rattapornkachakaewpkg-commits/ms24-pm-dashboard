## 🏆 เกณฑ์การตัดสินการแข่งขัน

### รอบภายในศูนย์ (Internal)
- คะแนนทฤษฎี: 40%
- คะแนนปฏิบัติ: 60%
- เกณฑ์ผ่าน: 60% ขึ้นไป
- จำนวนผู้ผ่าน: ศูนย์ละ 1-2 คน

### รอบภูมิภาค (Regional)
- คะแนนทฤษฎี: 30%
- คะแนนปฏิบัติ: 50%
- คะแนนนำเสนอ/สัมภาษณ์: 20%
- เกณฑ์ผ่าน: 70% ขึ้นไป
- จำนวนผู้ผ่าน: ภาคละ 3-5 คน

### รอบประเทศ (National)
- คะแนนทฤษฎี: 25%
- คะแนนปฏิบัติ: 50%
- คะแนนนำเสนอ/สัมภาษณ์: 15%
- คะแนน Teamwork: 10%
- เกณฑ์ผ่าน: 75% ขึ้นไป
- รางวัล: ชนะเลิศ, รองชนะเลิศ, รองชนะเลิศอันดับ 2

## 📝 Checklist การเตรียมตัว

### สำหรับช่าง
- [ ] อ่านคู่มือ 4JJ1, 4JK1, RZ4E ครบ
- [ ] ฝึกใช้ OBD-II Scanner ได้คล่อง
- [ ] ฝึกใช้ Multi-meter วัดค่าต่างๆ
- [ ] ฝึกถอด-ประกอบเครื่องยนต์/เกียร์
- [ ] ทำ Mock Exam 60 ข้อ ผ่าน 80%
- [ ] ฝึกจับเวลาปฏิบัติงาน

### สำหรับครูฝก
- [ ] เตรียม Presentation 15 นาที
- [ ] สร้างสื่อการสอน (PowerPoint/Video)
- [ ] ฝึกสอน Workshop จริง
- [ ] สร้างข้อสอบวัดผล
- [ ] ฝึกใช้ Simulator/E-Learning platform

### สำหรับ Sales Advisor
- [ ] จำ Spec รถอีซูซุทุกรุ่นได้
- [ ] ฝึก Test drive presentation
- [ ] ฝึกตอบข้อโต้แย้งลูกค้า
- [ ] จำจุดเด่น vs คู่แข่งได้
- [ ] ฝึก Role-play ขายรถ

### สำหรับช่างสี-ตัวถัง
- [ ] ฝึกพ่นสีได้เรียบเนียน
- [ ] ฝึกผสมสีได้ตรง
- [ ] ฝึกดึง-เคาะตัวถัง
- [ ] ฝึก Estimate ความเสียหาย
- [ ] ฝึกจับเวลาปฏิบัติงาน

### สำหรับ Service Advisor
- [ ] ฝึกต้อนรับลูกค้า 5 Steps
- [ ] ฝึกสร้าง RO ได้เร็ว
- [ ] จำอาการเสียที่พบบ่อยได้
- [ ] ฝึก Upselling/Cross-selling
- [ ] ฝึกจัดการข้อร้องเรียน

## 💡 คำแนะนำเพิ่มเติม

1. **เริ่มเตรียมตัวล่วงหน้า** อย่างน้อย 2-3 เดือน
2. **ฝึกปฏิบัติจริง** สำคัญกว่าทฤษฎี
3. **ทำ Mock Exam** อย่างน้อย 3 ครั้งก่อนแข่ง
4. **ศึกษาคู่มือ Isuzu** ให้ละเอียด
5. **ฝึกจับเวลา** ให้ชินกับเวลาแข่งขัน
6. **ดูแลสุขภาพ** นอนพักผ่อนให้เพียงพอ
