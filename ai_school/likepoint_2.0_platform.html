Key Features

ฟีเจอร์หลัก 10 อย่าง + Merchant Admin 8 อย่าง

💸
1. โอน (Transfer)
โอน Point ระหว่างผู้ใช้ได้ง่าย สะดวก รวดเร็ว
🎪
2. สร้างกิจกรรม
สร้างกิจกรรม Reward และ Promotion ได้ตามต้องการ
🔄
3. แลกกิจกรรม
ผู้ใช้สามารถแลก Point เป็นรางวัลต่างๆ ได้
🏆
4. อันดับ (Ranking)
จัดอันดับผู้ใช้ สร้างการแข่งขันและ Engagement
🔒
5. POI Lock
ล็อค Point ไว้รับรางวัลพิเศษเมื่อครบเงื่อนไข
💵
6. Cash-out
ถอน Point เป็นเงินสดหรือโอนเข้าบัญชีธนาคาร
🎰
7. Lotto (สลาก)
ระบบสลากลุ้นรางวัล สร้างความตื่นเต้นให้ผู้ใช้
🎡
8. Spin Wheel V2
วงล้อหมุนรางวัล เวอร์ชัน 2 ควบคุมงบประมาณได้
📦
9. Asset Claim
ระบบเคลม Asset และรางวัลอัตโนมัติ
🔐
10. Lock Plan (กำลังมา)
แผนล็อค Point ระยะยาว รับสิทธิพิเศษเพิ่ม
🚀
Upcoming Feature: TAP2EARNฟีเจอร์ใหม่ที่กำลังจะมา — แตะหน้าจอเพื่อรับ Point สร้าง Engagement แบบ Tap-to-Earn

️ Merchant Admin Dashboard Features

1. Dashboard
ภาพรวมกิจกรรม POI, จำนวนผู้เข้าร่วม, ยอด Point
2. POI Reports
รายงานกิจกรรม POI ทั้งหมด
3. Pocket Reports
รายงานกระเป๋า Point แต่ละคน
4. Bookbank History
ประวัติการทำรายการทั้งหมด
5. Features Config
ตั้งค่าฟีเจอร์ต่างๆ ของ Platform
6. Create POI / LOCK
สร้างกิจกรรม POI และ Lock Plan ใหม่
7. Create Game **
สร้างเกม Spin Wheel, Lottery
8. Create Plan **
สร้าง Lock Plan และ Reward Plan
04

วิเคราะห์ AL, BP, LL, II

สิ่งที่ควรปรับปรุง, สิ่งที่ทำได้ดี, สิ่งที่ล้มเหลว, และไอเดียเพื่อปรับปรุง

📊 AL — วิเคราะห์ปรับเปลี่ยนแปลง
  • ปรับปรุง SPIN GAME เวอร์ชันใหม่ ให้สามารถควบคุมงบประมาณ และสามารถ Customize ได้มากขึ้น
  • เพิ่มระบบ Authen ของ platform
  • ออกแบบเกม / รูปแบบการแจก point เป็นตัวเลือกให้ Partner มากขึ้น
✅ BP — สิ่งที่ทำได้ดีและต้องทำต่อ
1. การเพิ่ม Feature Toggle
  • BU สามารถเปิดปิดส่วนที่มีปัญหาได้ทันที
  • BU สามารถสื่อสารกับลูกค้าได้ ก่อนตัดสินใจเปิด Features
  • สามารถ Cut off และเปิดงานพัฒนา โดยไม่ Conflict กับ PM BU
2. การสื่อสารการนำ Feature / API ไปใช้งานบนแอป BU มากขึ้น
  • เข้าถึง Features ได้สะดวกมากขึ้น
  • แก้ปัญหา UX เรื่องการกดหลายระดับชั้นเพื่อเข้าถึง Features ใน Minilike
⚠️ LL — สิ่งที่ล้มเหลว
  • การสื่อสาร Auth จาก BuApp อย่างเดียวไม่พอ ต้องพัฒนา Auth ของ Bupoint เอง
  • Cash-out จ่ายเกินไปในเชิงเทคนิค เป็นช่องทำให้เกิดผลเสียหาย
💡 II — ไอเดียเพื่อปรับปรุง
  • Re-design เพื่อเพิ่ม UX ที่ดีขึ้น ปรับ Design ช่วง Q1/2026
  • ออกแบบ Game เพื่อเป็นตัวเลือกให้กับ BU เลือกเปิดใช้งาน
  • เพิ่มระบบ Authen ของ Bupoint

สรุปความต้องการ

NEW!! SPIN WHEEL & coming Features

🎡 1. ALgorithm — SPIN WHEEL

  • ควบคุมงบประมาณ % rate + Gachapon Pool กรอกลงงบประมาณและจำนวนรางวัลที่เพิ่มเข้ามา
  • ควบคุมงบประมาณได้
  • ทุก users ที่เข้ามาหมุนมีรางวัลแน่นอน
  • การเพิ่มรางวัลที่นอกเหนือจาก Point, คูปอง, เสื้อ หมวก รับ
  • Custom Spin ได้มากขึ้น (พื้นหลัง, point, icon)
  • from PVP NB (ให้ดูสนุกสนานขึ้นเล่นกัน Animation)

🔐 2. Lock Plan

  • Setup Config ให้ ยืดหยุ่น
  • BU สามารถกำหนดว่าให้ Point เพิ่มกำหนดจำนวนวัน
  • BU สามารถกำหนด Privilege เช่น Lotto Ticket +2, Spin round +2

💰 3. Bupoint Loan

  • ใช้ระบบเบื้องหลัง CU, Features จะอยู่บน Likepoint Platform แต่เปิดใช้งานผ่าน MS24
  • แปลงจากระบบ Lockpoint ให้ Admin CU เป็นผู้ดูแล Pocket ส่วน หลักประกัน
