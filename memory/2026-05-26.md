- **ไฟล์:** `/home/admin/.openclaw/workspace/ai_school/ai_app_builder_handbook.html`
- **ชื่อ:** คู่มือสร้าง App ด้วย AI 100% — จากศูนย์สู่ Prototype พร้อม Deploy
- **กลุ่มเป้าหมาย:** ผู้ใช้ AI ที่ไม่อยากเขียนโค้ด แต่อยากสร้าง App เอง
- **ระยะเวลา:** 4-8 สัปดาห์ (แล้วแต่ความซับซ้อน)
- **ค่าใช้จ่าย:** ฟรี - $20/เดือน

### 📋 5 Phases ในแผนฝึก:
| Phase | สัปดาห์ | เนื้อหา | เครื่องมือหลัก |
|:---:|:---:|:---|:---|
| 1. วางแผน | 1-2 | เขียน Spec, User Flow, Database Schema, เลือก Tech Stack | ChatGPT/Claude |
| 2. ออกแบบ | 2-3 | สร้าง Wireframe/Mockup, UI Design | v0.dev/Lovable/Uizard |
| 3. สร้าง Prototype | 3-6 | Database, Frontend ทุกหน้า, Routing, Auth, เชื่อมระบบ | Supabase, Lovable |
| 4. ทดสอบ | 6-7 | Test Cases, แก้ Bug, User Testing | Manual + AI |
| 5. ส่ง Deploy | 7-8 | Documentation, แพ็กเกจ 10 อย่าง, Kick-off Developer | Google Drive |

### 🎯 หลักสูตรนี้มี:
- Prompt สำหรับ AI ทุกขั้นตอน (Copy ไปวางได้เลย)
- Checklist แบบ Interactive (กดเช็คได้)
- Progress Bar แสดงความคืบหน้า
- ตารางสรุปเครื่องมือทั้งหมด + ราคา
- Data Flow Diagram เห็นภาพตั้งแต่ต้นจนจบ

### 📦 แพ็กเกจที่ส่งให้ Developer (10 อย่าง):
1. โค้ดทั้งหมด (GitHub/ZIP)
2. Database Schema + ข้อมูลตัวอย่าง
3. Documentation
4. Requirement Document
5. User Flow Diagram
6. UI Design (ถ้ามี)
7. Test Cases + ผลทดสอบ
8. รายการ Bug ที่ยังค้าง
9. Environment Variables
10. วิดีโอสาธิตการใช้งาน

| 🔒 Security | 6-7 | OWASP Top 10, HTTPS, bcrypt, PDPA, RBAC, MFA |
